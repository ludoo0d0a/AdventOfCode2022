# AdventOfCode2022

https://adventofcode.com/2022/

# Get data of the day

```bash
node init.js init 
````

or for a specific day
```bash
node init.js init -day 6 
````

# Submit answer

```bash
node init.js submit -answer 123456
````

or for a specific day
```bash
node init.js submit -day 6 -answer 123456
````
